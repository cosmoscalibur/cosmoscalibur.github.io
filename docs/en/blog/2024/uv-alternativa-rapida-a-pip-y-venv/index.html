UV: A Fast Alternative to PIP and VENV#

UV is a package manager and virtual environment alternative for Python, developed in Rust, which promises to be very fast in the processes it aims to replace those of PIP and VENV. In this sense, all you need to do is add uv before the usual instructions, and it should work (except for some particular cases of compatibility or lack of implementation).

This is one of many tools that are emerging with Rust to permeate the Python ecosystem, generating a new value, such as improved performance. And from the same creator, we also have Ruff as an efficient replacement for linters, and other tools like PyO3 for creating Python libraries in Rust, Polars as an alternative to Pandas, or Robyn as an alternative to Flask.

This project, UV, aims to be the equivalent of a cargo (the Rust package manager) for Python, integrating multiple utilities into a single binary.

In general, it‚Äôs a good practice to use environments and UV will do so by default. However, in case of a global installation, we can pass --system as an argument (suitable for CI or containers).
