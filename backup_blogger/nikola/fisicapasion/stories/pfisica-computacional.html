Lenguajes de Programación


C y C++
Compilador: GCC (Linux/Unix) y MinGW (Windows). Se recomienda instalar en Windows Cygwin.
Editores: CodeBlocks, editor de C++ multiplataforma y de código abierto; y Qt, ambiente de desarrollo multiplataforma de C++. Permite la creación de aplicaciones GUI usando un framework gráfico para asistencia visual de diseño.
Fortran
Compiladores: gfortran, ; g95, compilador estable de fortran 95 para multiples arquitecturas y sistemas operativos. Partes del estándar de fortran 2003 han sido implementados.
Editores
M
Lenguaje interpretado de alto nivel. Particularmente orientado al manejo de matrices. Este lenguaje surge de MathWorks con la plataforma MatLab.
Editores e interpretes: Octave, alternativa libre para el lenguaje de MatLab. Existen algunas pequeñas diferencias entre el lenguaje M de MatLab y Octave, pero la mayoria de scripts son portables; QtOctave, interface gráfica para octave desarrollada en Qt.
Python
Lenguaje interpretado de alto nivel. Una alternativa a MatLab (Lenguaje M). Se recomienda instalar las librerias de Numpy (libreria numerica) y Scipy (libreria de herramientas cientificas), Matplotlib y Mayavi2 (librerias de visualizacion).
Interpretes: Python,
Editores: Spyder, IDE Python multiplataforma fuertemente recomendado, si usa windows se recomienda la versión incluida en Enthought Python Disribution; Pyragua, entorno de desarrollo para la manipulación de código de lenguaje Python; Boa, IDE Python multiplataforma y asistente visual de diseño de wxpython.

Algebra Computacional
Sage
Software de matemáticas libre de código abierto licenciado bajo GPL usando una interface basada en Python.
Maxima
Software de álgebra computacional libre para la manipulación de expresiones simbólicas y numéricas incluyendo diferenciacion, integración, series de Taylor, transformadas de Laplace, ecuaciones diferenciales ordinarias, sistemas de ecuaciones lineales, polinomios, conjuntos, listas, arreglos, matrices y tensores. Permite graficar funciones en 2 y 3 dimensiones. Interface gráfica sugerida WxMaxima.
